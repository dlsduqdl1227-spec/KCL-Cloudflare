# KCL Stage71 표준편차·켈리브레이션 편차 기준 전면 업데이트

## 요청 사항
- 표준편차/점수 차이 판정 기준이 러프하게 잡혀 있던 문제 수정.
- 켈리브레이션 화면과 검수 표준편차 화면에서 1점을 초과한 차이가 `양호`로 표시되지 않도록 전면 업데이트.

## 적용 기준
전 대회 공통으로 표준편차와 헤드 기준 대비 차이 판정 기준을 아래처럼 통일했습니다.

| 구간 | 표시 |
|---|---|
| 0.00 ~ 1.00점 | 양호 |
| 1.01 ~ 2.00점 | 주의 |
| 2.00점 초과 | 점검 |

핵심 원칙: `1.00점 이내`만 양호이며, `1.00점 초과`는 절대 양호로 표시하지 않습니다.

## 수정 파일
- `public/assessment/index.html`

## 수정 내용
1. 공통 편차 판정 함수 추가
   - `scoreSpreadBadge_()` 추가
   - `scoreSpreadGuideHtml_()` 추가
   - `stddevBadge_()`와 `ikrcCalBadge_()`가 공통 기준을 사용하도록 변경

2. 일반 검수 / 표준편차 화면 기준 강화
   - 기존 총점 기준 양호 ≤3 또는 더 큰 값, 개별항목 기준 양호 ≤0.5/0.7 등으로 분산되어 있던 기준을 제거
   - 모든 표준편차 판정을 `양호 ≤1.00 / 주의 ≤2.00 / 점검 >2.00`으로 통일

3. MOB 켈리브레이션 화면 보강
   - 표준편차뿐 아니라 헤드 기준 대비 평균 차이(`헤드차`, `평균-헤드`)에도 동일 기준 배지 적용
   - 1점을 초과한 헤드차는 양호가 아닌 `주의` 또는 `점검`으로 표시

4. IKRC 켈리브레이션 화면 보강
   - 기존 Total/개별항목의 서로 다른 판정 기준 제거
   - 센서리 평균 ± 표준편차, 켈리브레이션 표 모두 공통 기준 적용
   - 헤드 기준 평가가 있는 경우 평균-헤드 차이도 같이 표시하고 동일 기준으로 판정

5. 화면 안내 문구 업데이트
   - 켈리브레이션/표준편차 영역에 `양호 ≤1.00 / 주의 ≤2.00 / 점검 >2.00` 안내 추가
   - `1점을 초과하면 양호로 표시하지 않습니다.` 문구 추가
