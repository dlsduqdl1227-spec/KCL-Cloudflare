# TEST REPORT - STAGE26 PDF NAME VISIBILITY FINAL

## 목적
전체 대회 순위/PDF 출력에서 선수명이 비공개로 남는 위험을 점검하고 수정했습니다.

## 확인된 문제
Stage25 기준으로 심사위원 화면은 블라인드가 정상 적용되지만, KCR / IKRC 예선 / KCAC 예선처럼 심사위원 제출 payload에 선수명이 비어 있는 경우, 팀장/관리자 순위 PDF에서도 선수명이 비어 보일 수 있는 구조였습니다.

## 수정 내용
1. `functions/api/rpc.js`
   - 순위/디브리핑/최종정리파일 생성 시 `participants` 테이블의 번호 매핑을 다시 조회하도록 보강했습니다.
   - `출품번호 / 샘플번호 / 블라인드번호 / 팀번호 / 참가자번호`를 라운드별 번호와 매칭합니다.
   - 관리자/대회팀장 결과 화면에서는 KCR·IKRC·KCAC 예선도 번호와 선수명이 함께 표시됩니다.
   - 심사위원 평가 화면의 블라인드 표시는 유지됩니다.
2. `public/assessment/index.html`
   - 순위 행에 `선수명`과 `소속`이 함께 표시되도록 보강했습니다.
   - 최종정리 엑셀의 순위 시트/점수 시트에 `소속/팀` 열이 반영되도록 수정했습니다.
