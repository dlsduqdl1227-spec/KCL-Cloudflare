데이터 초기화
선택한 대회 1개 기준으로만 초기화합니다. 전체 초기화는 지원하지 않습니다. 선수 초기화는 선수 등록 데이터만 삭제하고, 운영계정 초기화는 해당 대회 심사위원·팀장·운영진 권한만 삭제합니다. 점수 초기화는 해당 대회의 평가·검수·순위·최종 엑셀 반영 데이터만 삭제하므로 대회 시작 전 테스트 데이터 정리에만 사용하세요. 다른 대회 데이터와 관리자 계정은 삭제되지 않습니다.
전체 엑셀 등록
엑셀 파일 하나로 모든 대회의 선수 탭과 운영계정 탭을 한 번에 등록합니다. 엑셀 템플릿은 기존 원본 양식 그대로 사용하고, 시스템이 대회별로 필요한 번호열만 읽어 내부 등록값으로 변환합니다.
엑셀 템플릿 다운로드
읽는 탭: 각 대회별 _선수, _운영계정 탭 · 선수 시트는 대회별로 필요한 열만 들어간 정리 양식을 사용합니다. 블라인드 대회는 심사위원 화면에 번호만 표시됩니다.
