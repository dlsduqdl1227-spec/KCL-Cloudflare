# TEST REPORT — STAGE18 SELF REVIEW FINAL

## 수정 목적
Stage17에서 심사위원 검수 접근을 과하게 막아, 심사위원 본인이 제출한 평가까지 검수 목록에서 확인/검수완료 처리할 수 없는 문제가 있었습니다. 이번 Stage18에서는 권한을 다음처럼 재정리했습니다.

- 관리자: 전체 검수 목록, 전체 검수 수정, 순위, 최종정리파일 다운로드 가능
- 대회팀장: 담당 대회 전체 검수 목록, 전체 검수 수정, 순위, 최종정리파일 다운로드 가능
- 심사위원/운영진: 본인이 제출한 평가만 검수 목록에서 확인 및 검수완료 처리 가능
- 심사위원/운영진: 순위 조회와 최종정리파일 다운로드는 불가

## 구현 변경 사항

1. `getReviewList`
   - 기존: 관리자 또는 대회팀장만 접근 가능
   - 변경: 대회 접근 권한이 있는 계정은 접근 가능
   - 단, 관리자/대회팀장이 아니면 `judge_name` 또는 제출 payload의 심사위원 식별정보 기준으로 본인 제출 건만 반환

2. `updateReviewRow`
   - 관리자/대회팀장: 기존처럼 전체 검수 수정 가능
   - 심사위원/운영진: 본인이 제출한 행만 수정/검수완료 가능
   - 검수완료된 항목을 미검수로 되돌리는 작업은 관리자/대회팀장만 가능

3. `updateReviewStatus` / `updateReviewStatusBatch`
   - 관리자/대회팀장: 전체 상태 변경 가능
   - 심사위원/운영진: 본인 제출 건만 `검수완료` 가능
   - 타 심사위원 제출 건 상태 변경 차단

4. 평가 화면 UI
   - 일반 심사위원/운영진에게도 `내 제출 검수` 버튼 표시
   - 관리자/대회팀장에게는 기존처럼 `검수 / 현황 보기` 표시
   - 순위 버튼과 최종정리파일 다운로드는 관리자/대회팀장 권한 유지
