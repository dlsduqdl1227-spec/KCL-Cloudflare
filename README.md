# KCL Assessment Cloudflare v2 V7

Cloudflare Pages + Functions + D1 기반 KCL 심사/검수/순위/디브리핑 웹앱입니다.

## 주요 페이지
- `/` : KCL Assessment 메인
- `/assessment/` : 평가/관리자
- `/debriefing/` : 선수 디브리핑 인증/조회
- `/registry/` : 선수·심사위원 등록 및 블라인드 매핑 관리

## V7 핵심
- 규정집 기준 대회별 순위/동점 기준 보강
- 선수 사전 등록 및 블라인드 번호 매핑
- 디브리핑 인증을 선수 등록 데이터와 점수 unit 매칭으로 연결
- 브라우저 인쇄 기반 PDF 저장

자세한 적용 방법은 `docs/SETUP_KR.md`와 `README_V7_업데이트_가이드.txt`를 참고하세요.
