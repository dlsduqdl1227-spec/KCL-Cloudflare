# KCL Assessment 1.0ver Security Stage 14

- 관리자 로그인 UI는 아이디 / 비밀번호 / 시크릿 코드 3단계 입력을 유지합니다.
- 관리자 시크릿 코드 기본값은 `5061`이며, Cloudflare Pages Secret `KCL_ADMIN_SECRET_CODE`로 변경할 수 있습니다.
- 등록·권한 관리 화면의 중복 초기화 버튼을 하나의 `데이터 초기화` 접이식 영역으로 통합했습니다.
- 전체 엑셀 등록 영역을 탭 밖의 공통 카드로 분리하여 선수·운영·관리자 탭 구조를 단순화했습니다.
- 모바일 환경에서 탭은 가로 스크롤형 버튼으로 표시되며, 주요 버튼과 입력 필드는 화면 폭에 맞게 재배치됩니다.
- 관리자 계정은 관리자 탭에서 이름 / 핸드폰번호 / 소속으로 추가할 수 있으며 전체 관리자 권한으로 저장됩니다.
- 운영계정 초기화 시 관리자 계정은 삭제 대상에서 제외됩니다.
- 최종 점검 결과는 `TEST_REPORT_STAGE14.md`를 참고하세요.
