# KCL Assessment Web App - Stage47

Stage47은 Stage46 MOB 역할별 검수 항목 매칭 보강본을 기반으로, MOB 켈리브레이션 검수 운영 흐름을 정리한 최종 후보입니다.

핵심 변경:
- MOB 헤드 켈리브레이션 목록의 `확인` 버튼 명칭을 `검수확인`으로 변경
- MOB 켈리브레이션 상세 화면에 `검수완료` 버튼을 별도 배치
- `검수완료` 처리한 MOB 켈리브레이션 항목은 목록에서 숨김 처리
- MOB 헤드 기준 평가는 기존처럼 순위·총점 집계에서 제외 유지
- MOB 센서리/테크니컬 역할별 항목 분리 구조 유지
- 전체 화면 스크립트 및 서버 코드 정적 검사 완료

사용 파일: `kcl-assessment-1.0ver-security-stage47-mob-calibration-complete-final.zip`
