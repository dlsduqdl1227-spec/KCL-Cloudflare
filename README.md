# KCL Assessment Stage43

Stage43는 Stage42 기능을 기준으로 다음 운영 안정화 항목을 추가한 버전입니다.

## 주요 변경

1. 대회별 점수 데이터 초기화
- 등록/권한 관리 화면의 데이터 초기화 영역에 `점수 데이터 초기화` 버튼 추가
- 선택한 대회의 평가 점수, 검수 상태, 순위 산출 데이터 삭제
- 선수 등록 데이터와 운영계정은 유지
- MOB 켈리브레이션 확인 데이터, IKRC 켈리브레이션/Seed to Cup 보조 데이터 정리 포함

2. MOB 테크니컬 점수 표시 개선
- 테크니컬 평가항목 점수와 시간감점을 분리 표시
- 총평가 반영점수를 별도 표시
- 평가 화면 상단, 하단 점수 반영 기준, 검수 목록, 검수 상세, 순위 상세, 선수 디브리핑에 반영

3. 역할별 검수/디브리핑 항목 매칭
- MOB 테크니컬 심사위원: 테크니컬 항목, 시간 관련 항목, 테크니컬 총점/총평가 반영 표시
- MOB 센서리 심사위원: 센서리 및 창작메뉴 관련 항목 중심 표시
- 브루잉 라운드에서 창작메뉴 항목이 노출되지 않도록 필터 보강

4. 스크롤 안정성 보강
- 평가, 검수, 디브리핑 화면에서 세로 스크롤이 막히지 않도록 CSS 가드 보강
- 모바일/작은 화면에서 상단 고정 버튼과 본문 영역이 충돌하지 않도록 유지

## 배포 참고

- Cloudflare Pages에 업로드 후 `/registry/`에서 관리자 로그인하여 기능 확인
- 대회 직전 테스트 데이터 정리 시 `점수 데이터 초기화` 사용
- 실제 대회 데이터가 들어간 뒤에는 초기화 버튼 사용 전 반드시 입력 문구 확인 절차를 통과해야 함
