# TEST REPORT · STAGE36 REGISTRY CRUD CLEAN FINAL

## 목적
관리자 등록·권한 관리 화면을 대회 운영 기준에 맞게 정리하고, 등록된 선수/팀·심사위원·팀장·운영진·관리자 목록에서 직접 추가/수정/삭제가 가능하도록 보강했습니다.

## 반영 내용

### 1. 선수·팀 등록 탭 정리
- 기존 `선수·팀 등록` CSV/선택 대회 엑셀 불러오기 카드 제거
- 상단 `전체 엑셀 등록`만 일괄 등록 기능으로 유지
- 선수·팀 개별 추가 폼 유지
- 등록된 선수·팀 목록에서 `수정` / `삭제` 버튼 추가
- `수정` 클릭 시 개별 등록 폼에 기존 데이터가 채워지고, 저장 시 기존 행을 갱신하도록 처리
- `삭제` 클릭 시 확인 후 해당 선수·팀 등록 데이터만 삭제
- 삭제 시 점수 데이터는 삭제하지 않음

### 2. 운영 권한 탭 정리
- 기존 선택 대회 운영계정 CSV/엑셀 개별 일괄 등록 카드 제거
- 상단 `전체 엑셀 등록`만 일괄 등록 기능으로 유지
- 심사위원·팀장·운영진 개별 추가 폼 유지
- 등록된 심사위원·팀장 목록에서 `수정` / `삭제` 버튼 제공
- `수정` 클릭 시 이름, 연락처, 소속, 계정유형, 역할, 평가팀이 폼에 채워지고 기존 행을 갱신

### 3. 관리자 탭 보강
- 관리자 목록에 `수정` / `삭제` 버튼 제공
- 관리자 수정 시 아이디/비밀번호/소속 수정 가능
- 마지막 관리자 계정 삭제 차단 서버 로직 유지

### 4. 안전 처리
- 대회 선택 변경 시 편집 중 상태 자동 해제
- 수정 취소 버튼 추가
- 운영계정/관리자 삭제 실패 시 메시지 표시 후 목록 유지
- 선수·팀 삭제 실패 시 메시지 표시 후 목록 유지
- KTCC 팀원정보 등 기존 extra 데이터는 수정 시 보존되도록 처리
