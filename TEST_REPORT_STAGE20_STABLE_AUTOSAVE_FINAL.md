# TEST REPORT — STAGE20 STABLE AUTOSAVE FINAL

## 적용 목적
대회 현장에서 무료 Cloudflare 환경을 안정적으로 사용하기 위해 두 가지를 보강했습니다.

1. KCAC 라떼아트 이미지 저장 용량 안정화
2. 평가 중 휴대폰 전원 꺼짐/브라우저 종료/새로고침 발생 시 직전 작성 내용 이어쓰기

## 구현 내용

### 1. KCAC 이미지 자동 압축
- 사진 업로드 시 원본을 그대로 보관하지 않고 1280px 이하, JPEG 품질 0.72 기준으로 1차 압축합니다.
- 가이드 합성 스냅샷 저장 시 960px 이하로 재생성합니다.
- 최종 저장용 full 이미지는 450KB 이하를 목표로 품질과 해상도를 자동 조정합니다.
- 썸네일은 180px, 80KB 이하를 목표로 저장합니다.
- 서버 저장 payload에는 최신 스냅샷 2개까지만 포함하여 D1 row size 초과 위험을 낮췄습니다.
- KCAC 패턴평가 제출 시 스냅샷 미저장 상태면 기존처럼 제출을 차단합니다.

### 2. 평가 자동 임시저장
- 평가 화면에서 input/change/click 발생 시 localStorage에 자동 임시저장합니다.
- 브라우저 종료 직전, 앱 전환/화면 꺼짐 직전에도 한 번 더 저장합니다.
- 다음 접속 후 같은 심사위원/대회/라운드 평가 화면에 들어가면 이전 임시저장 복원 여부를 묻습니다.
- 복원 대상:
  - KCR: 컵별 점수, 강도, 태그, 코멘트, 현재 컵/항목
  - KCAC: 잔별 점수, 리프 수, 패턴 선택, 감점, 코멘트, 압축 스냅샷
  - KBC: 참가자 정보, 점수, 시간, 코멘트, Y/N, 태그
  - MOB: 참가자 정보, 점수, 메뉴, 시간, 코멘트, 태그
  - IKRC: 컵별 점수, 강도, 태그, 코멘트, 현재 컵/항목
  - MOC: 참가자별 타깃, 정답 선택, 국가/농장/프로세스, 시간, 검수 상태
  - KTCC: 팀별 타깃, 섹션별 정답 개수, 시간, 테이블/팀 정보
- 제출 성공 시 해당 임시저장은 자동 삭제됩니다.

## 운영 메모

- 자동 임시저장은 사용자의 기기 브라우저 localStorage에 저장됩니다. 같은 휴대폰/같은 브라우저에서 이어쓰기 가능합니다.
- 다른 기기로 바꿔 접속하면 localStorage가 공유되지 않으므로 이어쓰기는 되지 않습니다.
- KCAC 이미지는 DB 안정성을 위해 원본 사진이 아니라 압축된 기록용 스냅샷 중심으로 저장됩니다.
- 대회 시작 전 테스트 데이터는 반드시 초기화한 뒤 운영을 시작하세요.
