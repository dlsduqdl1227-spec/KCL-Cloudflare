# KCL Stage69 MOB 총점 합산 기준 수정

## 수정 배경
Stage68에서 MOB 순위 집계 설명과 일부 계산 기준에 `센서리 평균`, `카테고리별 평균 합산`이라는 표현과 로직이 들어가 있었습니다. 2026 MOB 룰북 기준으로 순위는 총점 기준이며, 동점 시 예선은 센서리 점수 → 테크니컬 점수 → 경기시간, 결선은 센서리 점수 → 테크니컬 점수 → 창작 메뉴 점수 → 경기시간 순입니다.

## 수정 내용
1. MOB 실시간 순위 집계
   - 기존: 센서리 평균 + 테크니컬 평균 + 창작 평균 - 시간감점
   - 수정: 센서리 총점 합산 + 테크니컬 총점 합산 + 창작메뉴 총점 합산 - 시간감점

2. 시간감점 반영
   - 10분/15분 기준 초과에 따른 시간감점은 참가자 공식 총점 합산에서 1회 차감합니다.
   - 감점 적용 후 음수인 경우 순위 반영점수는 0점으로 제한합니다.

3. 화면 문구 수정
   - `센서리 평균` → `센서리 총점`
   - `창작메뉴 평균` → `창작메뉴 총점`
   - `카테고리별 평균 합산` → `총점 합산`

4. 중복 제출 방어
   - 같은 심사위원/같은 역할/같은 참가자에 중복 제출이 존재하면 최신 행만 공식 집계에 반영합니다.
   - 상세 계산 박스에 원본 행 수와 공식 반영 행 수가 다르면 표시합니다.

## 예시
예선에서 센서리 18, 21.6, 23 / 테크니컬 14.8 / 시간감점 40점이면:

- 감점 전 합산: 18 + 21.6 + 23 + 14.8 = 77.4
- 시간감점: -40
- 최종 순위 반영점수: 37.4

실격은 아니며 0점 처리 대상도 아닙니다.
