# KCL Stage74 — KCAC 사진 전용 미디어 입력 최종 수정

## 요청 요약
KCAC 제출 화면에서 영상 파일은 용량이 커서 운영 안정성에 부담이 될 수 있으므로, `앨범에서 영상 불러오기` 기능을 제거하고 우선 사진 중심으로 운영한다.

## 최종 적용 정책
KCAC 미디어 입력은 아래 2개 방식만 지원한다.

1. 촬영하기
2. 앨범에서 사진 불러오기

영상 파일 업로드/프레임 추출은 지원하지 않는다.

## 수정 내용

### 1. KCAC 평가 화면 `public/assessment/index.html`
- `앨범에서 영상 불러오기` 버튼 제거
- `video/*` 파일 입력 제거
- `onKcacVideoInput()` 함수 제거
- `album-video` 분기 제거
- `videoMeta` 저장 필드 제거
- 영상 길이 관련 최종 제출 extra 필드 제거
- 안내 문구를 사진 전용 기준으로 변경

현재 KCAC 평가 화면 버튼 구조:

```text
촬영하기
앨범에서 사진 불러오기
가이드 함께 저장
현재 사진 제거
```

### 2. KCAC 컵 가이드 화면 `public/camera/index.html`
- `앨범에서 영상 불러오기` 버튼 제거
- 영상 파일 입력 제거
- 영상 프레임 추출 함수 제거
- 영상 관련 DOM 참조 제거
- 안내 문구를 사진 전용 기준으로 변경

현재 KCAC 컵 가이드 화면 미디어 입력 구조:

```text
촬영하기
앨범에서 사진 불러오기
카메라 중지
```

## 유지된 기능
- 카메라 촬영
- 앨범 사진 불러오기
- 가이드선 표시
- 가이드 위치 조정
- 가이드 함께 저장
- 저장된 스냅샷 확인
- KCAC 평가 제출 payload 내 사진 스냅샷 유지

## 제거 이유
영상 원본은 파일 용량이 크고, 모바일 브라우저에서 프레임 추출 시 기종/브라우저별 오류 가능성이 있다. 대회 운영 안정성을 우선하여 사진 전용 구조로 단순화했다.
