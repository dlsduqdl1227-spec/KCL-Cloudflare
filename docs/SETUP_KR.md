# KCL Cloudflare v2 V7 설정

## 1. 배포
GitHub에 V7 파일을 업로드하면 Cloudflare Pages가 자동 배포합니다.

필수 교체 파일:
- `functions/api/rpc.js`
- `public/assessment/index.html`
- `public/debriefing/index.html`
- `public/registry/index.html`

## 2. D1 마이그레이션
Cloudflare D1 Console에서 `migrations/0003_v7_registry.sql`을 실행합니다.

## 3. 선수/심사위원 등록

## 4. 블라인드 대회 등록 원칙
- KCR: 선수명/연락처 + 컵번호 또는 출품번호 등록
- IKRC: 선수명/연락처 + 샘플번호 등록
- KCAC: 선수명/연락처 + 참가자번호 등록
- MOC: 선수명/연락처 + 참가자번호 등록
- KTCC: 팀명/연락처 + 팀번호 등록

평가 입력은 블라인드 번호로 진행하고, 디브리핑 조회는 선수명과 연락처로 인증합니다.
V7은 등록된 매핑을 이용해 블라인드 점수와 선수 본인을 연결합니다.

## 5. PDF
V7은 서버 PDF 생성 대신 브라우저 인쇄 기능을 사용합니다.
디브리핑 화면에서 PDF 다운로드 버튼을 누른 뒤, 인쇄 창에서 `PDF로 저장`을 선택합니다.

## 6. 진행도
V7 적용 후 기능 기준 진행도는 약 84%입니다.
남은 핵심 작업은 실제 SMS 발송 연동, 서버 PDF 자동 생성, 대회별 실데이터 전체 검증입니다.
