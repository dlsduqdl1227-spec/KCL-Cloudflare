# KCL 디지털 평가 웹앱 객관 검수 기록 — Stage67

## 검수 기준
- 2026 KCL 7개 대회 룰북 기준으로 점수 산식, 검수 상태, 실격, 시간감점, 켈리브레이션 제외 로직을 재확인했습니다.
- 코드 문법 검사는 `functions/api/rpc.js`, `functions/api/health.js`, 각 HTML 내 인라인 스크립트 전체에 대해 수행했습니다.
- 실제 운영 DB 접속 및 실제 브라우저 클릭 테스트는 이 파일에서 수행하지 않았습니다.

## 이번 Stage67에서 추가 수정한 사항
1. 최종정리 엑셀 다운로드가 `검수완료`만 포함하던 부분을 `검수완료 + 수정완료` 모두 포함하도록 수정했습니다.
   - 순위 집계 기준과 최종정리 파일 기준이 다르면, 검수에서 수정 저장한 점수가 순위에는 반영되지만 최종 엑셀에는 누락될 수 있어 보정했습니다.
2. 최종정리 엑셀 안내 문구를 `검수완료·수정완료 포함 / 미검수·켈리브레이션 제외`로 변경했습니다.
3. 디브리핑 화면의 상태 배지가 `수정완료`도 공식 검수 완료 상태로 보이도록 수정했습니다.

## 재확인한 핵심 로직
- 순위 반영: 검수완료, 수정완료
- 순위 제외: 미검수, 켈리브레이션
- MOB/IKRC 헤드 기준 데이터: 순위 제외
- MOB 시간감점: 예선 10분, 결선 15분 기준 / 1~15초 -6점 / 16~30초 -24점 / 31~60초 -40점 / 1분 초과 실격
- MOB 경기시간 파싱: `10분 08.00초`, `10.08`, `10:08.00` 모두 608초로 처리
- MOB 음수 시간감점: 가산되지 않고 양수 감점으로 처리
- KCR: Sweetness x2 반영
- KCAC: 2잔 합산, 리프수 감점과 시간감점 분리 반영
- 실격: 순위 점수 0점 유지, 관리자/팀장 명시 수정 시 해제 가능
