## 4. 남은 버그 및 운영 제한

현재 로컬 재현 기준으로 순위나 저장을 잘못 계산하는 미수정 코드 버그는 확인되지 않았다. 아래는 현장 투입 전 반드시 이해해야 할 운영 제한이다.

1. **IKRC Technical Time**: 장비 이상 접수, 승인/불승인, 타이머 정지·재개·재시연을 자동 처리하는 전용 기능은 없다. 외부 공식 기록지를 사용할지 운영진 확정이 필요하다. 외부 기록이 없다면 IKRC 운영 전 기능 추가가 필수다.
2. **KCR 정규화**: 시스템은 입력·검수된 `정규화점수/공식점수/순위반영점수`를 순위에 우선 사용하지만, 정규화 공식을 자동 산출하지 않는다. 공식 산식과 입력 책임자를 확정해야 한다.
3. **초기화 범위**: 전체 대회 초기화는 차단되어 있고 대회별 초기화만 가능하다. 라운드 한 개만 초기화하는 기능은 제공하지 않으므로 실제 점수가 들어간 뒤에는 초기화 버튼을 사용하지 말고 관리자 검수에서 개별 수정/삭제해야 한다.
4. **PDF**: 서버가 PDF 파일을 생성하는 방식이 아니라 브라우저 인쇄 창에서 `PDF로 저장`을 선택하는 방식이다. 행사 노트북의 인쇄/PDF 저장 기능을 사전 확인해야 한다.
5. **KCAC 미디어**: 영상 파일 직접 업로드는 용량 안정성을 위해 지원하지 않고 제출 URL을 저장한다. 평가 사진은 모바일에서 압축 저장한다. 실제 영상 링크 권한과 사진 촬영/앨범 권한을 행사 기기로 확인해야 한다.
6. **엑셀 라이브러리**: 브라우저가 XLSX CDN을 불러오지 못하면 호환 `.xls` 방식으로 내려받는다. 행사 네트워크에서 실제 `.xlsx` 다운로드를 확인해야 한다.
7. **원격 환경**: 이번 검수는 로컬 Cloudflare Pages/D1에서 수행했다. 운영 중인 원격 D1과 SOLAPI 키를 사용한 실검증은 수행하지 않았다.
