## 템플릿 확인

- ZIP 루트에도 동일한 원본 템플릿 1개만 포함.
- SHA-256 동일성 확인 완료.

## 등록 로직 확인
- KBC/MOC/MOB: 참가자번호 중심, 현장 즉시 입력 및 사전등록 모두 가능.
- KCR: 예선컵번호/컵번호를 예선 출품번호로 사용, 결선컵번호를 결선 출품번호로 사용. 심사위원 화면은 블라인드 번호 표시.
- IKRC: 샘플번호/예선컵번호를 예선샘플번호로 사용. 예선 심사위원 화면은 블라인드 처리.
- KCAC: 예선컵번호/컵번호를 예선블라인드번호로 사용, 결선컵번호를 결선참가번호로 사용. 본선컵번호/팀명은 무시.
- KTCC: 원본 양식처럼 팀원별 2행으로 등록해도 팀번호·팀명 기준으로 1팀으로 병합하고 팀원 정보는 extra_json에 보존.
