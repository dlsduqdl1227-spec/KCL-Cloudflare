KCL Assessment 1.0ver - 다음 단계 업데이트

적용 핵심
1. 대회팀장 겸임 권한 수정본을 유지합니다.
2. 순위 산정 시 동점 처리 보조값 계산 방식을 보강했습니다.
   - MOB: 센서리/테크니컬/창작메뉴 동점 비교값을 첫 항목이 아니라 관련 항목 합산으로 계산
   - KCAC 결선: 센서리 합산, 디자인 관련 항목 합산으로 계산
   - KCAC 예선: 패턴 완성도, 패턴 균형 비교 유지
   - KBC: 총점 동점 시 에스프레소 합산 → 프레젠테이션 → 머신 운용 순 비교값 추가
   - KCR/IKRC/MOC/KTCC 기존 규정 흐름 유지
3. 검수완료된 점수만 순위에 반영되는 구조를 유지합니다.
4. 실격 데이터는 순위표에 실격으로 노출되도록 유지합니다.

교체 권장 파일
- functions/api/rpc.js

전체 교체 시 주의
- wrangler.toml은 기존 GitHub의 정상 파일을 유지하세요.
- database_id가 REPLACE_WITH_YOUR_D1_DATABASE_ID로 바뀌면 배포가 실패합니다.

테스트 순서
1. GitHub에서 functions/api/rpc.js 교체
2. Cloudflare Deployments Success 확인
3. 관리자 로그인
4. 각 대회별 테스트 점수 2~3개 검수완료
5. 동일 점수 2개를 만들어 동점 처리 순서가 깨지지 않는지 확인
6. 순위 상세에서 점수/코멘트가 보이는지 확인
