KCL Assessment 1.0ver - 관리자 환경변수/전체권한 최종 보강

수정 내용
- KCL_ADMIN_NAME / KCL_ADMIN_PHONE 환경변수가 실제 관리자 정보로 설정되어 있을 때, 같은 이름/연락처의 대회팀장 행이 이미 있어도 전체관리자(ALL/ADMIN) 행을 별도로 생성하도록 수정했습니다.
- 기존 개발용 기본값(관리자 / 01000000000)은 코드에서 완전히 무시합니다.
- 로그인 권한 병합 기준을 이름+연락처 일치로 정리해, 기본 관리자 또는 다른 이름의 행이 섞여 권한을 오판하지 않도록 보강했습니다.

핵심 교체 파일
- functions/api/rpc.js

적용 후 권장 작업
1. Cloudflare 환경변수 KCL_ADMIN_NAME / KCL_ADMIN_PHONE이 실제 관리자 정보인지 확인합니다.
2. 기본값이 관리자 / 01000000000이면, 실제 이름/연락처로 덮어쓰거나 D1에 ADMIN / ALL 행을 직접 추가합니다.
3. 배포 후 /admin/에서 다시 로그인 / 권한 새로고침을 누르고 다시 로그인합니다.

주의
- wrangler.toml은 기존 정상 파일을 유지하세요.
